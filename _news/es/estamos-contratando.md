Estamos buscando personas desarrolladoras de software tiempo parcial (part-time) para fortalecer nuestro equipo técnico.

En esta oportunidad requerimos dos perfiles para trabajar en el desarollo de nuestra plataforma de cosecha y enriquecimiento de metadatos, agregado de metadatos de sistemas CRIS (Current Research Information System), deduplicación de entidades, implementación de dashboards para repositorios y herramientas de visualización de indicadores/métricas.

### **[](https://www.lareferencia.info/es/blog-masonry/noticias/item/305-estamos-contratando)[Perfil JAVA/SPRING]**

Profesional con experiencia en el desarrollo de software para trabajar en proyectos relacionados con nuestra plataforma abierta de cosecha y enriquecimiento de metadatos, agregado de metadatos de sistemas CRIS, de duplicación de entidades .

Buscamos las siguientes habilidades y conocimientos técnicos:

Imprescindibles

*   Conocimiento profundo de Java y Spring Framework / Spring Boot.
*   Experiencia la interacción con aplicaciones web y desarrollo de servicios RESTful.
*   Conocimiento de bases de datos relacionales (preferentemente PostgreSQL)
*   Buenas habilidades de comunicación y capacidad para documentar y explicar soluciones técnicas.

Importantes:

*   Conocimiento profundo de Java y Spring Framework, incluyendo Spring Boot y Spring MVC.
*   Experiencia con motores de búsqueda (Solr / Elasticsearch)
*   Experiencia en integración de aplicaciones con diferentes sistemas y plataformas.
*   Conocimiento de Git y metodologías ágiles
*   Experiencia en el uso de herramientas de construcción y automatización como Maven o Gradle.
*   Buen manejo de idioma inglés

Valorables (no excluyentes)

*   Conocimiento de HTML, CSS, JavaScript y frameworks front-end como Angular o React.
*   Conocimiento de la plataforma DSpace y su ecosistema sería un plus.
*   Experiencia en la integración de sistemas y en la implementación de protocolos de interoperabilidad, como OAI-PMH y OAI-ORE.
*   Experiencia en el desarrollo de servicios RESTful y en la integración con bases de datos relacionales y NoSQL.

### **[](https://www.lareferencia.info/es/blog-masonry/noticias/item/305-estamos-contratando)[Perfil Web UI/Data Viz]**

Profesional con experiencia en el desarrollo de software para trabajar en proyectos relacionados con nuestra plataforma abierta de cosecha y enriquecimiento de metadatos enfocado en el desarrollo de interfaces web, dashboards, herramientas de visualización, procesamiento de datos textuales y estadísticos.

Buscamos las siguientes habilidades y conocimientos técnicos:

Imprescindibles

*   Conocimiento de HTML, CSS, JavaScript, Typescript en frameworks front-end como Angular o React (o similares).
*   Experiencia en el desarrollo de aplicaciones web interactuando con servicios RESTful.
*   Capacidad para trabajar con APIs y datos dinámicos.
*   Buenas habilidades de comunicación y capacidad para documentar y explicar soluciones técnicas.

Importantes:

*   Conocimiento profundo de Angular o React (o similar) y sus respectivas bibliotecas y herramientas complementarias.
*   Experiencia en la visualización de datos, creación de dashboards y gráficos estadísticos.
*   Experiencia en integración de aplicaciones con diferentes sistemas y plataformas.
*   Conocimiento de Git y metodologías ágiles
*   Buen manejo de idioma inglés

Valorables (no excluyentes)

*   Experiencia con motores de búsqueda (Solr / Elasticsearch)
*   Experiencia en el desarrollo de servicios RESTful y en la integración con bases de datos relacionales y NoSQL.
*   Conocimiento de Python y sus librerías para análisis de datos como NumPy, Pandas y Matplotlib.
*   Experiencia en la manipulación y limpieza de grandes conjuntos de datos.
*   Conocimiento de algoritmos y técnicas estadísticas para el análisis de datos.
*   Experiencia en el uso de librerías para el procesamiento de lenguaje natural como NLTK o spaCy.
*   Conocimiento de tecnologías de aprendizaje profundo como TensorFlow o PyTorch.
*   Conocimiento de modelos de procesamiento de lenguaje natural como el modelo BERT.
*   Buenas habilidades matemáticas y estadísticas para comprender los conceptos subyacentes a la inteligencia artificial y el aprendizaje profundo

Ambos perfiles requieren una formación en informática (o carrera relacionada) y al menos 3 años de experiencia en desarrollo de software.
