A versão 7.0 do DSpace já está disponível. **Esta versão apresenta uma nova interface de usuário (Angular) que reúne o melhor das interfaces JSPUI e XMLUI, retrabalhando todas as funcionalidades do DSpace.** A nova interface de usuário é suportada por uma nova API REST, que expõe todos os conteúdos e funcionalidades, **permitindo a integração e interoperabilidade do DSpace com sistemas e serviços externos como nunca antes.**

Esta versão reúne toda a comunidade DSpace sob uma única interface de usuário, permitindo-nos progredir juntos mais rapidamente.

"O DSpace 7.0 é o maior lançamento da nossa história, com mais de um milhão de linhas de código do DSpace alteradas, mais do que os últimos 4 lançamentos anteriores combinados. **Apesar da pandemia global, aceleramos nosso desenvolvimento e lançamos um total de 5 versões Beta, culminando em um Testathon global.** Este lançamento é o resultado da dedicação, determinação e comprometimento de nossa comunidade, que alimenta continuamente o programa DSpace." Kristi Park, Presidente de Governança do DSpace e CEO da Biblioteca Digital do Texas

**DSpace 7.0 foi desenvolvido pela colaboração de mais de 60 desenvolvedores da comunidade, liderados por LYRASIS, Atmire e 4Science. Mais de 25 instituições em todo o mundo forneceram apoio financeiro generosamente.** Veja a lista completa de colaboradores nas [notas de versão](https://wiki.lyrasis.org/display/DSDOC7x/Release+Notes).notas de versão. Se você gostaria de contribuir para o próximo lançamento 7.x, considere participar de futuras reuniões do [Grupo de Trabalho DSpace 7](https://wiki.lyrasis.org/display/DSPACE/DSpace+7+Working+Group).

**Recursos do DSpace**

[DSpace versão 7.0](https://duraspace.org/dspace/download/) inclui os seguintes recursos:

* Uma nova interface de usuário baseada em Angular (substituindo XMLUI e JSPUI). O objetivo da nova interface é implementar todos os principais recursos de XMLUI e JSPUI em uma interface de usuário única e moderna.
* Uma API REST nova e completa, usando as melhores práticas modernas. O objetivo da nova API REST é implementar uma arquitetura moderna, segura, independente e escalável.
* Submissões e fluxos de trabalho redesenhados com um processo de envio de uma página, interface de arrastar e soltar e uma reformulação do MyDSpace.
* Um novo modelo de objetos Entidade Configuráveis, que permite a criação de novas tipologias de entidades, bem como contemplar as relações entre elas. Este recurso permite maior integração com sistemas de identificação externos (por exemplo, ORCID), sistemas CRIS, sistemas de publicação de periódicos eletrônicos, etc.
